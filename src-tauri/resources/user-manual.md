# SoundBatcher ユーザーマニュアル

このドキュメントは、利用者向けの基本操作に絞ったマニュアルです。

## 1. 画面構成

- 左: ファイルブラウザ
- 中央: 波形プレーヤー
- 右: Processor / History パネル
- 上部ツールバー: マニュアル、設定

## 2. ファイルを選んで再生する

1. 左側のファイルブラウザでフォルダを開きます。
2. ファイルをクリックすると選択できます（複数選択可）。
3. ファイルをダブルクリックすると再生と波形表示を開始します。

## 3. バッチ処理を実行する

1. 処理対象のファイルを選択します。
2. 右側の `Processor` タブで `Add Selected` を押します。
3. キューに入ったことを確認して `Start Processing` を押します。

## 4. 処理結果を確認する

- 処理済みファイルは入力元フォルダの `processed` に保存されます。
- `History` タブから、過去の実行結果とエラー内容を確認できます。

## 5. 設定

ツールバーの `設定` から以下を変更できます。

- 出力先（同一フォルダ / カスタムフォルダ）
- ファイル名（接頭辞 / 接尾辞）
- 既存ファイル時の挙動（リネーム / 上書き / スキップ）
- 出力形式、ラウドネス正規化、同時実行数
- 言語、テーマ

## 6. トラブルシュート

- 処理が失敗する場合は、`History` の詳細ログを確認してください。
- 入力ファイルが読み取り専用や破損状態だと失敗する場合があります。
- FFmpeg のサイドカーが正しく配置されているかも確認してください。
