# SoundBatcher ユーザーマニュアル

このドキュメントは、利用者向けの基本操作に絞ったマニュアルです。

## 1. 画面構成

- 左: ファイルブラウザ
- 中央: 波形プレーヤー
- 右: 処理 / 履歴 パネル
- 上部ツールバー: マニュアル、設定
- ファイルブラウザ上部: タブバー（追加、切替、並び替え、固定）

## 2. ファイルを選んで再生する

1. 左側のファイルブラウザでフォルダを開きます。
2. ファイルをクリックすると選択できます（`Ctrl/Cmd` で追加選択、`Shift` で範囲選択）。
3. ファイルをダブルクリックすると再生と波形表示を開始します。
4. 戻る/進むは `Alt+←` / `Alt+→`、またはブラウザ上部のボタンで操作できます。
5. `Backspace` でも1階層戻れます。

## 3. バッチ処理を実行する

1. 処理対象のファイルを選択します。
2. 右側の `処理` タブで `選択を追加` を押します。
3. キューに入ったことを確認して `処理開始` を押します。
4. 右クリックメニューから `キューに追加` / `選択項目をキューに追加` でも登録できます。

英語UIでは、`処理` は `Proc`、`選択を追加` は `Add Selected`、`処理開始` は `Start Processing` と表示されます。

## 4. 処理結果を確認する

- 処理済みファイルは入力元フォルダの `processed` に保存されます。
- `履歴` タブから、過去の実行結果とエラー内容を確認できます。
- ファイル一覧は定期的に自動更新され、アプリに戻ったタイミングでも再読込されます。

## 5. 設定

ツールバーの `設定` から以下を変更できます。

- 出力先（同一フォルダ / カスタムフォルダ）
- ファイル名（接頭辞 / 接尾辞）
- 既存ファイル時の挙動（リネーム / 上書き / スキップ）
- 実行モード（波形処理 / リネームのみ）
- 出力形式、ラウドネス正規化、同時実行数
- 言語、テーマ

## 6. ファイルブラウザ便利操作

- タブをドラッグすると並び替えできます。
- タブ右クリックで固定/解除、タブクローズ、他タブクローズができます。
- フォルダ右クリックで開く、ピン留め、配下音声のキュー追加、新しいタブで開くができます。
- ファイル右クリックで再生、キュー追加、パスコピーができます。
- 外部からフォルダをドロップするとそのフォルダを開きます。
- 外部から音声ファイルをドロップすると親フォルダを開いて対象を選択します。
- ファイルをフォルダへドラッグすると移動できます（同一フォルダ内は移動しません）。

## 7. トラブルシュート

- 処理が失敗する場合は、`履歴` の詳細ログを確認してください。
- 入力ファイルが読み取り専用や破損状態だと失敗する場合があります。
- FFmpeg のサイドカーが正しく配置されているかも確認してください。
